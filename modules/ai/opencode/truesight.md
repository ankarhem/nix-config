# Truesight: Semantic Code Search and Repo Structure Visualization

Use `truesight_search_repo` when:

- Conceptual/semantic queries ("how is X handled?", "retry patterns", "authentication flow")
- Don't know exact file, class, or method names
- Need ranked, relevant results across the codebase

Use `truesight_repo_map` when:

- Need to understand module boundaries and structure
- Want dependency hints and key symbols
- Search has narrowed the area and you need architectural context

Use `truesight_index_repo` when:

- Search results seem missing or stale
- New files added but not appearing in search
